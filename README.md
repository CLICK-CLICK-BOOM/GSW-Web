# Game Store Wrangler

Game Store Wrangler is a free utility designed to manage Game Store applications.

**IF YOU DON'T WANT GAME STORES TO BE ALLOWED TO RUN WHEN YOU ARE NOT USING/NEEDING THEM, TRY GSW.**

Add the stores you want it to manage and away it goes. It features a number of detailed controls for fine-tuning to your exact needs. It has a small install footprint and is easier to operate than it looks—5 minutes and you are set up and working. 

It is the sort of utility you can just leave running quietly in your system tray once you know it's working. (A second monitor is great for watching it do its thing while you are still getting a feel for it!)

---

### Important Notes
* **Admin Permissions:** The app requires administrator privileges to interact with and manage OS process loops.
* **Code Signing:** Please note that the executable is currently unsigned. Digital code signing is a target for a future update.
* **Safe & Non-Destructive:** GSW will not delete files or mess up your PC. It is completely non-destructive; at very worst, it simply crashes.
